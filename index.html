semantic-KERN

This is a learning and development project that explores how the KERN UX standard can be used as a lightweight starter kit that supplies sensible styles by default.

Features

  • Focuses on styling Markdown content: lists, headings, paragraphs.
  • Can be used as a standalone style, or together with KERN-native.

    .custom
    will disable these styles.
